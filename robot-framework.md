## Robot Framework
一个很优秀的“关键字”驱动的框架。（后续继续介绍）

## 安装
需要先配置Python2.7,pip  
```
pip install framework
```
或者使用源码安装，
```
python setup.py install
```
安装成功即可。

## RIDE
[RIDE](https://github.com/robotframework/RIDE) is a development environment for Robot Framework test cases.   

** 安装 **
```
pip install robotframework-ride
```
或者使用源码安装，

```

python setup.py install

```
